# Dashboard Features (V1)

## Daily Command Center
- Top 3 priorities
- Current blockers
- Quick actions (sync canvas, morning brief, run scout)

## Growth Pipeline
- experiments[] from `mustang_context.json`
- status: idea / queued / running / result / adopted

## Outreach Queue
- list from `outreach_queue[]`
- fields: segment, status, confidence, last_contacted

## Build & Ship
- GitHub PR/issues snapshot (next iteration)
- CI health

## Skills Inventory
- output of `scripts/list_skills.sh`

## Job Monitor
- parse `logs/job_runs.jsonl`
- show latest run per job + failures
